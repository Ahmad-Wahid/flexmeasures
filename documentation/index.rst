Welcome to A1 VPP's documentation!
==================================

A1 VPP is a multi-user web application for a virtual power plant, which can monitor and steer a diverse set of assets. The system allows aggregators and asset owners to monitor power loads and generation as well as relevant price data. In addition, the system uses algorithms for machine learning and control strategies to assist human operators with forecasts and decision-making.
